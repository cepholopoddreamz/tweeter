# Tweeter Project

Tweeter is a simple, single-page Twitter clone. This project is a first attempt to work with Jquery, CSS, and Ajax . 

## Dependencies

- Express
- Node 5.10.x or above

### Getting Started: 

Go to http://localhost:8080/ in your browser.
